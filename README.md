# Insta Clone

A full-stack Instagram clone built using the **MERN stack**.  
Users can create an account, log in, create posts, upload images, like posts, and interact with other users.

---

## Features

- User authentication (Register / Login / Logout)
- Create posts with caption and image
- Like posts
- View user profile
- JWT based authentication
- Image upload support
- Backend API with Express and MongoDB
- Frontend built with React

---

## Tech Stack

### Frontend
- React
- Vite
- CSS / Tailwind / SCSS 

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Cookie Parser
- Multer / ImageKit
